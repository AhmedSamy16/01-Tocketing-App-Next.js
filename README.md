# 01-Ticketing-App-Next.js

This is project 01/50 of my Next.js projects series,
it's a simple Ticketing app that allows you to store tickets based on their category, priority and their status.

## Stack used

- Next.js
- Typescript
- React Hook Form
- TailwindCSS
- MongoDB
